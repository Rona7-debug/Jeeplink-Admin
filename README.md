# JeepLink Admin — Admin Panel

The administrative side of the JeepLink application, 
designed to manage jeepney route data, fare structures, 
and system content for the JeepLink commuter app.

## About

JeepLink Admin allows authorized administrators to manage 
and maintain the data that powers the JeepLink mobile app, 
including jeepney routes, fare rates, and user data 
oversight.

## Features

- Admin authentication and secure access control
- Manage jeepney routes and route codes
- Update fare rates and distance data
- Manage user data and preferences
- Firebase security rules — admin-only write access
- Read-only public data management

## Built With

- Kotlin
- Android Studio
- Firebase Authentication
- Cloud Firestore
- Firebase Security Rules
